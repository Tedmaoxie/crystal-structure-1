# CRYSTAL VIS - 晶体结构教学可视化系统

**AI赋能高中化学教学系列 by 实验中学**

## 📖 项目简介

CRYSTAL VIS 是一个专为高中化学教学设计的 Web 端 3D 晶体结构可视化工具。本项目利用 WebGL 技术（基于 Three.js），将抽象的微观晶体结构转化为直观的 3D 模型，帮助学生理解晶胞结构、原子堆积方式以及复杂的配位关系。

系统支持从基础晶胞到复杂的离子晶体、分子晶体及混合型晶体的展示，并特别针对教学场景开发了“配位数点亮系统”和“教研演示模式”。

## ✨ 核心功能

### 1. 多样化晶体库
支持 18 种典型高中化学晶体结构模型，涵盖：
*   **基础晶胞**：简单立方、体心立方 (BCC)、面心立方 (FCC)
*   **离子晶体**：氯化钠 (NaCl)、氯化铯 (CsCl)、氟化钙 (CaF2)、氧化亚铜 (Cu2O)、硫化锌 (ZnS)
*   **金属晶体**：铜 (Cu)、钠 (Na)、锌 (Zn)
*   **共价晶体**：金刚石 (C)、晶体硅 (Si)
*   **分子晶体**：碘 (I₂)、干冰 (CO₂)、冰 (H₂O)
*   **混合/拓展**：石墨 (C)、钙钛矿 (CaTiO₃)

### 2. 交互式配位分析系统
*   **智能配位识别**：点击任意原子，系统自动计算并高亮其配位原子（Coordination Number, CN）。
*   **3D 空间扩展**：自动搜索 3x3x3 晶胞空间，寻找晶胞外的邻近原子，完整复原配位环境。
*   **视觉增强**：
    *   **高亮与暗化**：点亮配位原子的同时，自动暗化非相关原子，突出观察重点。
    *   **动态连线**：在中心原子与配位原子间生成连线。
    *   **分子晶体优化**：针对 I₂、CO₂ 等分子晶体，支持“点击分子”交互，并在晶胞外使用“质点”模型简化显示，避免视觉杂乱。
    *   **动态标签**：实时跟随原子的 "CN=X" 悬浮标签。

### 3. 教研演示模式
*   一键开启自动巡航，系统自动切换晶体结构。
*   **全自动教学流**：全览旋转 -> 自动选取代表性原子演示配位 -> 恢复全览 -> 切换下一个。
*   非常适合课堂演示或展屏循环播放。

### 4. 沉浸式视觉体验
*   **Glassmorphism UI**：现代化的毛玻璃风格界面。
*   **动态效果**：晶胞呼吸特效、冰晶体中氢原子的热振动模拟、氢键与共价键的区分显示。

## 🛠️ 技术栈

*   **HTML5 / CSS3**
*   **JavaScript (ES6+)**
*   **[Three.js](https://threejs.org/)** (WebGL 3D 引擎)

## 🚀 快速开始

本项目为纯前端项目，无需配置复杂的后端环境。

1.  **克隆或下载**本项目到本地。
2.  直接双击打开 `index.html` 文件（推荐使用 Chrome, Edge, Firefox 等现代浏览器）。
3.  系统加载完成后即可开始交互。

## 🎮 操作指南

*   **旋转视角**：按住鼠标左键拖拽。
*   **缩放视角**：滚动鼠标滚轮。
*   **平移视角**：按住鼠标右键拖拽。
*   **查看配位**：点击晶胞中的任意原子（或分子）。再次点击或点击空白处可取消高亮。
*   **切换结构**：使用底部控制栏的“下一个”按钮或左侧导航栏。
*   **演示模式**：点击顶部标题栏右侧的“教研演示模式”开关。

## 📝 开发与维护

*   所有逻辑均包含在 `index.html` 中，便于单文件分发和部署。
*   如需修改晶体数据，请编辑代码中的 `structures` 数组配置。
