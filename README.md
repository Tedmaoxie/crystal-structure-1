# Crystal Structure Visualization System (晶体结构可视化系统)

这是一个基于 Three.js 的交互式晶体结构可视化工具，旨在辅助化学教学和科研展示。该系统支持多种典型晶体结构的 3D 展示、配位数分析、晶胞堆积演示等功能。

## 🌟 主要功能 (Features)

*   **多模式展示**: 支持多种晶体结构的 3D 渲染，包括金属晶体（简单立方、体心立方、面心立方、六方最密堆积）、离子晶体（NaCl, CsCl, ZnS, CaF2）、共价晶体（金刚石、石墨、SiO2）、分子晶体（干冰、冰）等。
*   **交互式操作**:
    *   **360° 旋转/缩放**: 自由观察晶体结构细节。
    *   **配位数探测**: 点击原子可自动高亮其配位原子，并显示配位数信息。
    *   **暗化模式**: 专注模式下可暗化背景原子，突出显示选中的结构。
    *   **晶胞切换**: 支持在单晶胞和 2x2x2 堆积模式间切换。
*   **教研演示模式 (Demo Mode)**:
    *   自动轮播展示所有晶体结构。
    *   智能展示流程：全览 -> 自动拆解/高亮不同种类原子的配位情况 -> 全览 -> 下一个结构。
    *   适合课堂教学或展厅演示。
*   **科学可视化**:
    *   **真实渲染**: 采用物理材质 (PBR) 渲染原子光泽。
    *   **动态键合**: 区分共价键和氢键的显示样式。
    *   **LOD 优化**: 针对大规模粒子系统进行性能优化，保证流畅运行。

## 🚀 快速开始 (Getting Started)

### 本地运行
1.  克隆仓库:
    ```bash
    git clone https://github.com/Tedmaoxie/crystal-structure-1.git
    ```
2.  直接在浏览器中打开 `index.html` 文件即可运行（推荐使用 VS Code 的 Live Server 插件以避免跨域问题）。

### 在线部署 (GitHub Pages)
本项目已配置为支持 GitHub Pages，可直接在线访问（请在 Settings 中开启）。

## 🛠️ 技术栈 (Tech Stack)
*   **Three.js**: 核心 3D 渲染引擎。
*   **HTML5/CSS3**: 界面 UI 与布局。
*   **JavaScript (ES6+)**: 逻辑控制与交互实现。

## 📝 操作指南 (User Guide)
*   **左键拖拽**: 旋转视角。
*   **滚轮**: 缩放视角。
*   **右键拖拽**: 平移视角。
*   **点击原子**: 查看该原子的配位环境。
*   **控制面板**: 底部包含复位、暂停演示、切换暗化模式等功能按钮。
